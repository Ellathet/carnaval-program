# Carnaval Project
This is a application to learn a little more about [gin](https://github.com/gin-gonic/gin) and [gorm](https://gorm.io/index.html) ORM

# Install
Assuming you already have clone this repository, run a simple: 
```
go get
```

# Building
Build project:
```
make build
```

# Deploying
Start the api:
```
make start
```
